Frequently Asked Questions

Current Features & Operations

How do I create my first storyboard?

Select "Create New Storyboard," choose "Create Blank," pick an aspect ratio (like 16:9), and name your file to initialize the workspace. Saving the file first is required because the app organizes all frames and assets within a dedicated project folder.

How do I access the software settings?

Open Preferences via Storyboarder > Preferences (Mac) or Edit > Preferences (Windows), or use the shortcut Cmd/Ctrl + ,.

What should I do if the drawing performance is slow?

Disable the "High Quality Drawing Engine" in Preferences. This is recommended because it reduces the GPU rendering load, which is necessary for smooth performance on older graphics hardware.

Can I use other image editors like Krita or GIMP?

Yes, select your "External PSD Editor" in Preferences. This allows you to leverage advanced tools in other software while maintaining a seamless link to your storyboard frames.

How can I view or modify keyboard shortcuts?

View shortcuts via Help > Key Commands. To customize them, click "Reveal Keymap File" in Preferences and edit the keymap.json file in a text editor to remap functions to your preferred keys.

Is there a way to adjust frame timing?

Switch to "Timeline Mode" using the small "Boards" button in the bottom-left or via View > Toggle Boards/Timeline Mode. This view is essential for matching visual duration to audio cues precisely.

How do I stop annoying app notifications?

Uncheck "Show Hourly Affirmations" and "Allow Comments on Line Mileage" in the Preferences menu to remove non-essential alerts.

How do I use the Shot Generator?

Type a description in the sidebar to generate 3D layouts, then double-click the image to place it on your reference layer. This provides a structural 3D guide that you can trace over for accurate perspective.

Is my tablet or touch screen supported?

The app uses Electron/Chromium for input, so most Wacom tablets work well. If pen pressure fails, updating your drivers or disabling "Windows Ink" is recommended to resolve hardware communication conflicts.

How do I change the project frame rate?

Select your desired FPS from the "Frame Rate" dropdown in the right sidebar to ensure timing matches your final video output.

How do I import hand-drawn paper worksheets?

Either visit the IP address shown on your printed sheet using a mobile browser on the same Wi-Fi, or photograph the sheet and use File > Import worksheets... to automatically crop and align your sketches.

How do I add music or dialogue?

Select an audio file in the right sidebar or use the record button. Associating audio with the first board of a scene is recommended for long tracks, as the audio will play continuously across subsequent frames.

How do I remove the Scene IDs added to my script?

Delete the - #SCENENUMBER# tags in a Fountain script or use Production > Scene Numbers > Remove Numbers in Final Draft. Note that this will break the link between your script and storyboard files.

How do I zoom or move the canvas?

Use the scroll wheel to zoom (or Ctrl +/-) and hold Shift while dragging to pan. This allows for detailed drawing without altering the actual frame resolution.

Planned Roadmap

What features are coming soon?

Development is active for custom resolutions, marquee selection tools (move/scale/rotate), layer management, an iPad version, custom export templates, and straight-line drawing tools.

Will 3D and collaboration features be improved?

Future updates aim to include animated camera moves, 3D perspective grids, custom 3D model imports, and real-time team collaboration tools.

System Compatibility

Do you support Windows 7 or 32-bit systems?

No. The software requires 64-bit architecture and modern OS libraries to maintain performance and security standards, as these older environments are no longer supported by Microsoft or key development libraries.
